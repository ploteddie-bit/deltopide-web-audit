# Deltopide Web Audit — Chrome Web Store (EN)

## Summary (132 characters max)

SEO, GEO-AI & CBOR-Web audit in one click. Ultra-fast Rust/WASM engine. Instant scores, diagnostics and recommendations.

## Long description

Deltopide Web Audit is a professional Chrome extension that analyzes your web pages against current and emerging search visibility standards. Developed by the Franco-Spanish agency Deltopide, it combines classic technical SEO auditing, AI search engine visibility (GEO), and compatibility checking with the CBOR-Web standard.

The analysis engine is written in Rust and compiled to WebAssembly for optimal performance: a full page audit executes in milliseconds, entirely within your browser, with no data ever sent to an external server.

### Key features

- **Instant audit** — Full analysis in one click from the toolbar or side panel
- **Overall score 0-100** — Composite rating based on 4 weighted audit pillars
- **Detailed diagnostics** — Each criterion is evaluated with a clear status (Pass / Warning / Fail)
- **Actionable recommendations** — Concrete suggestions to fix every detected issue
- **Local history** — Track score evolution over time
- **Export results** — Reports ready for your clients or team
- **Zero data collection** — Everything stays in your browser

### The 4 audit pillars

**1. TECH — Technical foundations**
Core Web Vitals (LCP, FID, CLS), HTTPS, meta viewport, Open Graph, Twitter Cards, Schema.org structured data, loading performance.

**2. SEO — Content & authority**
Title and meta description tags, heading structure (H1-H6), image alt attributes, internal and external links, robots.txt, sitemap.xml, canonical tag.

**3. GEO — AI visibility**
Presence and quality of llms.txt file, content structuring for LLMs, clarity of potential answers, citations and sources, adaptation to AI answer engines.

**4. SXG — CBOR-Web**
Detection of index.cbor manifest, DNS TXT signature validation, publisher identity verification, compliance with the CBOR-Web standard for binary semantic web.
