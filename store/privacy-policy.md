# Privacy Policy — Deltopide Web Audit

**Last updated:** 2026-03-27

## Overview

Deltopide Web Audit is a Chrome browser extension developed by Deltopide SL. This extension performs web page audits entirely within your browser. We are committed to protecting your privacy: no personal data is collected, transmitted, or shared with any third party.

## Data collection

**Deltopide Web Audit does not collect any personal data.**

The extension does not send any information to external servers. All audit processing is performed locally in your browser using a Rust/WebAssembly engine. No network requests are made by the extension beyond what is strictly required to audit the current page (e.g., checking for the existence of robots.txt, sitemap.xml, llms.txt, or index.cbor on the audited domain).

## Permissions used

The extension requests the following Chrome permissions, each for a specific and limited purpose:

| Permission | Purpose |
|---|---|
| `activeTab` | Access the current tab's page content to perform the audit when the user clicks the extension icon |
| `sidePanel` | Display audit results in Chrome's side panel |
| `scripting` | Inject the content analysis script into the audited page to collect technical data (DOM structure, meta tags, headings, etc.) |
| `storage` | Save audit history and user preferences locally using `chrome.storage.local` |
| `<all_urls>` (host permission) | Allow auditing any website the user visits — the extension only activates when explicitly triggered by the user |

## Local storage

All data generated by the extension is stored locally on your device using `chrome.storage.local`. This includes:

- Audit results and scores
- Audit history for visited pages
- User preferences (display settings)

This data never leaves your browser. You can clear it at any time by removing the extension or clearing extension data in Chrome settings.

## What we do NOT do

- We do **not** track your browsing activity
- We do **not** use analytics or telemetry of any kind
- We do **not** set or read cookies
- We do **not** collect IP addresses or geolocation data
- We do **not** transmit any data to Deltopide servers or any third party
- We do **not** display advertisements
- We do **not** sell, share, or monetize any user data

## Third-party services

Deltopide Web Audit does not integrate with any third-party services, APIs, or analytics platforms. The extension is fully self-contained.

## Data security

Since no data is transmitted externally, there is no risk of data breach from our side. All audit data remains on your local machine under Chrome's extension sandboxing security model.

## Changes to this policy

If we update this privacy policy, we will publish the revised version with an updated date. Significant changes will be communicated through the Chrome Web Store listing.
